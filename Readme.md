## 使用方法

### 同文皮肤

##### *.trime.yaml文件

请直接放置在你的用户文件夹内，然后在设置中应用本皮肤。

### 无障碍版皮肤

##### *.trime.yaml文件

请直接放置在你的用户文件夹内，然后在设置中应用本皮肤。

##### *.rime文件

目前没有这种文件。

### 请注意

对于已经说明仅用于同文无障碍版的皮肤，请勿应用在原版同文上，避免出现严重问题。
